# network-scanning-tool
This tool was developed as my graduation project. It is a simple educational and experimental network scanning tool that combines both active and passive reconnaissance techniques.

The tool allows users to scan active devices on a network, perform port scanning, and monitor network activity in real time. Through monitoring mode, it can detect devices joining or leaving the network and generate alerts when potential Man-in-the-Middle (MITM) attacks are detected.

The application also includes an integrated database within the interface, enabling users to store and review previous scan results.

A packet analysis feature similar to Wireshark was originally implemented, but unfortunately the source file was lost and is therefore not included in the current version.
